# O projeto
Neste  Projeto  vamos  trabalhar  com  aprendizado  não supervisionado  para  segmentar clientes  por  similaridade.

# Objetivo
O  objetivo  é  encontrar  grupos  de  clientes  similares  e  assim  ajudar  a equipe de Marketing a compreender como orientar melhor as campanhas de acordo com o perfil dos clientes.
Usaremos dados fictícios.
